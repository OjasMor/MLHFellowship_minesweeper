Minesweeper Help

Instructions:

  • Your goal is to clear all tiles except the ones with mines
  • Clicking on a tile will reveal itself and any adjacent clear tiles
  • Flag tiles which you believe have mines located under them
  • Clicking on a tile with a mine will result in a lost game

 

Types of Tiles

Un-revealed Tile

This tile may contain nothing, a number, or a bomb underneath it.

Empty Tile

This tile is empty and contains no adjacent mines or bombs.

Flagged Tile

Flagging a tile means you believe a mine is underneath it. 

Number Tile

The number on the tile represents the number of bombs adjacent to this tile.

Bomb Tile

This tile is a bomb, if you are seeing this tile, you lost the game!
